Job Application Dashboard

The Problem

The job application process in itself can be thought of as a full-time job. Even the most organized job seekers can struggle keeping track of cover letters, company contacts, sent and received emails, and application deadlines. Many have tried to solve this problem with spreadsheets, folders, and note taking applications. However, the vast majority of job seekers are working on multiple applications at once while trying to keep track of applications that have already been submitted. In these situations, the materials pertaining to a specific application can be difficult to locate and failure to properly store this information in the proper location can cause serious problems.

The Solution

Job Application Dashboard is a comprehensive web application that is designed to help job seekers stay organized and to easily locate materials. The application considers the five categories associated with a job search:

  1. Reminders
  2. Jobs
  3. Contacts
  4. Communications
  5. Documents

This web application was designed to be “job centered.” For this reason, all other categories (i.e. reminders, contacts, communications, and documents) can be linked to a specific job posting. This is beneficial because all information related to a specific job post can be found in a single location

Category Details

Jobs

The jobs category of this application is the most important. This is where job posting information (e.g. job descriptions, requirements, and deadlines), is stored. It is also the location where all linked information can be found.

Reminders

The reminders category can be thought of as a to-do list. However, the job application process also requires job seekers to follow-up with potential employers. Therefore, a reminder can be created with a status of “pending.” This informs the user that no action is required, but some aspect of the job application should be monitored.

Contacts

The contacts category is where a user can store emails, phone numbers and other information pertaining to the representative of a potential employer.

Communications

Users have the option of using the communications category to store emails and notes related to a job posting. Linking communications to a job ensures that a user can quickly lookup emails and notes from previous conversations.

Documents

The documents category is where a job seeker can store materials that have been used with a job application. For instance, if a cover letter was written for a specific job application, it can be stored in this location for reference. This ensures that a user can quickly determine what information has been shared with a company.
